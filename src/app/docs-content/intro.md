# familiar

`familiar` is a conversation layer for executable systems.

It helps you put a clean conversational surface in front of tools, workflows, scripts, and services without forcing every integration to rebuild the same behavior.

## What it does

- accepts normalized text input
- keeps thread continuity
- remembers useful context
- asks follow-up questions when required details are missing
- chooses the right tool
- sends clean arguments to the executor that does the real work

## Who it is for

- junior developers who want a simple integration path
- product teams building tool-based chat experiences
- AI-built systems that need a small, predictable contract
