Try It

This example always goes through Texty. Type a normal user message, send it, and inspect the real Texty request and response below.

What This Example Does

It supports one tool: notes.echo. Texty decides whether to reply normally or call that tool.

  • The page uses a built-in demo executor id and demo user id.
  • The page syncs the example tool with Texty.
  • The page sends your message to Texty.
  • Texty decides whether to chat normally or call the tool.
  • If Texty calls the tool, the tool returns a structured result.
  • If the note is empty, it returns needs_clarification.
  • If the note has text, it returns completed.
  • If the token is wrong, it returns unauthorized.
