Minimal Texty Executor

This demo shows Texty driving a real external tool. Instead of hiding the effect inside API JSON, the example adds todos to a live sidebar so you can immediately see how the conversation changes state.

Running on __PORT__
Demo identity only
Tool: todos.add

Talk To Texty

Try messages like “add buy dog food to my todo list” or “remember to call mom tomorrow”.

The page always syncs the tool and sends your message through Texty. If Texty decides a tool should run, the external executor updates the todo list shown on the right.
