Minimal Texty Executor

This local page helps you test what Texty will send to your executor. Type a note, submit it, and you will see both the request payload and the JSON result.

Server running on http://localhost:__PORT__

Try It

This sends POST /tools/execute to this local server using the same shape Texty uses today.

What This Example Does

It supports one tool: notes.echo.

  • If the note is empty, it returns needs_clarification.
  • If the note has text, it returns completed.
  • If the token is wrong, it returns unauthorized.
